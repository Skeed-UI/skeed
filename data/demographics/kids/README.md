# Kids

> Apps for children ages 4–12. Learning, play, creativity, supervised social.

## Design language

| Axis | Choice |
|---|---|
| Palette | Candy pastels — pink #FF6B9D, yellow #FFD166, green #06D6A0, sky #4CC9F0 |
| Typography | Display: **Fredoka** rounded; Body: **Nunito**; Numeric: **Lexend** tabular |
| Density | `comfy` only |
| Radius | Up to 48px / pill |
| Motion | `playful` — bounce easings, 280–600ms |
| Iconography | `illustrated` — Open Doodles |
| Borders | 0 width, `soft` |
| Elevation | `paper` — pastel multi-layer shadows |
| WCAG | **AAA** floor |

## Forbidden patterns

This demographic carries strict ethics gates enforced by `@skeed/guards`. See [`psychology/learning.json`](./psychology/learning.json) for the full list. Highlights:

- No autoplay audio or video
- No infinite scroll
- No urgency timers, fake scarcity, or loss-aversion patterns
- No targeted advertising or third-party tracking by default
- No data collection without verifiable parental consent
- No social-comparison shaming (streak-shame, public leaderboards)
